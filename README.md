# setup

## requirements

|        |         |
| ------ | ------- |
| `node` | `>= 20` |
| `yarn` | `>= 1`  |

## installation

### install the project

1. clone repository

```sh
git clone <repository>
```

2. install `node` dependencies

```sh
yarn
```

### `nix`

Project can be run in a `nix-shell` environment.

```sh
nix-shell env.nix
```

---

# run

## development

Run the project locally in browser, project will be available at `http://localhost:8080`.

```sh
yarn start
```
